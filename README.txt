Программа загружает заказы из текстового файла, фильтрует их по заданному району и времени доставки, а затем сохраняет результаты в отдельный файл. Программа также ведет журнал операций для отслеживания всех шагов выполнения и возможных ошибок.

Файл заказов (Orders.txt) должен быть доступен в корневой папке проекта. Этот файл содержит данные о заказах, которые программа будет фильтровать.

Программа запросит следующие параметры:

Район доставки (например, DistrictA).
Время первой доставки в формате yyyy-MM-dd HH:mm:ss (например, 2024-10-29 12:15:00).


Структура файлов
Orders.txt: файл с заказами, которые будут фильтроваться.
deliveryOrder.txt: файл с результатами фильтрации.
deliveryLog.txt: лог-файл с записью выполненных операций.
